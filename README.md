# 🎬 SmartFlix – AI-Powered Movies App
 <img width="300" height="300" alt="logoo" src="https://github.com/user-attachments/assets/1cadf04d-005d-4869-afe3-bb6104c26f29" />

> **Your personal AI movie assistant** – Search, discover, and watch real-time top-rated movies with the power of AI & real APIs.

---

Features

- 🎥 **Real-Time Movies Data** – Powered by [TMDB API](https://www.themoviedb.org/), always up-to-date with the latest releases.
- 🤖 **AI Movie Search** – Don't know what to watch? Just ask! Our AI-powered search (via OpenAI API) will suggest the perfect movies.
- 🌐 **Multi-Language Support** – Switch between **English** and **Hindi** instantly.
- 📊 **Top Rated & Now Playing** – Curated lists so you never miss trending titles.
- 🔥 **Responsive & Modern UI** – Built with **React**, optimized for all devices.
- ☁ **Secure Auth & Storage** – Managed with **Firebase** for seamless sign-in/sign-up.

---

## 🛠 Tech Stack

| Technology   | Purpose |
|--------------|---------|
| **React.js** | Frontend UI |
| **Firebase** | Authentication & hosting |
| **TMDB API** | Real-time movies data |
| **OpenAI API** | AI-powered movie recommendations |
| **CSS / Tailwind** | Styling & responsiveness |

## 📦 Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/smartflix.git

# Navigate to the project folder
cd smartflix

# Install dependencies
npm install

# Start the development server
npm start

REACT_APP_TMDB_API_KEY=your_tmdb_api_key
REACT_APP_OPENAI_API_KEY=your_openai_api_key
REACT_APP_FIREBASE_API_KEY=your_firebase_api_key

🎯 How It Works
Fetch Movies – The app uses TMDB API to fetch now playing & top-rated movies.

AI Search – The OpenAI API interprets your query (e.g., "funny action movies") and returns suggestions.

Language Switch – Instantly change the UI language between Hindi and English.

Auth & Personalization – Users can sign in with Google or email via Firebase.
